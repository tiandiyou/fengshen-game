**版本**: v1.0

## 一、城池基础

### 1.1 城池定位

```
城池 = 玩家在大地图上的据点
每个玩家拥有1个主城
城池可升级/迁城
```

### 1.2 落地规则

```
出生时选择出生州
↓
随机分配到城池落点
↓
成为该城池领主
```

### 1.3 迁城规则

```
消耗资源可迁城
↓
迁城CD: 7天
↓
只能迁到同区域
↓
迁城后原城池变为空城
```

## 二、士气系统

### 2.1 士气机制

```
城池距离越近
↓
士气越高
↓
战斗伤害加成越高
```

### 2.2 距离计算

```
距离: 两个城池之间的格子数
士气: 100 - 距离 × 5

示例:
距离0 (同一城池): 100%士气
距离5: 75%士气
距离10: 50%士气
距离20: 0%士气
```

### 2.3 士气效果

```
士气影响战斗伤害:
实际伤害 = 基础伤害 × (士气 / 100)

示例:
100%士气: 伤害100%
75%士气: 伤害75%
50%士气: 伤害50%
```

### 2.4 最低士气

```
最低士气: 50%
最低伤害: 50%
```

## 三、城池建筑

### 3.1 建筑类型

| 建筑 | 功能 | 升级消耗 |
|------|------|----------|
| 城主府 | 决定其他建筑等级上限 | 金币 |
| 民居 | 产出金币 | 金币 |
| 粮仓 | 产出粮草 | 粮草 |
| 林场 | 产出木材 | 木材 |
| 灵田 | 产出灵气 | 灵气 |

| 仓库 | 提升资源储存上限 | 石头 |

### 3.2 建筑等级

| 建筑 | 1级 | 5级 | 10级 | 15级 | 20级 |
|------|-----|-----|-----|-----|-----|

| 民居 | 100/h | 500/h | 1000/h | 1500/h | 2000/h |

| 林场 | 100/h | 500/h | 1000/h | 1500/h | 2000/h |

| 校场 | +1000 | +5000 | +10000 | +15000 | +20000 |
| 仓库 | +5000 | +25000 | +50000 | +75000 | +100000 |

### 3.3 升级消耗

| 建筑 | 1→2级 | 2→3级 | ... | 19→20级 |
|------|-------|-------|-----|---------|
| 城主府 | 1000 | 2000 | ... | 19000 |
| 民居 | 500 | 1000 | ... | 9500 |
| 粮仓 | 500 | 1000 | ... | 9500 |
| 林场 | 500 | 1000 | ... | 9500 |
| 灵田 | 500 | 1000 | ... | 9500 |
| 校场 | 1000 | 2000 | ... | 19000 |
| 仓库 | 1000 | 2000 | ... | 19000 |

### 3.4 建筑等级上限

```
建筑等级上限 = 城主府等级

示例:
城主府5级 → 资源建筑最高5级
城主府10级 → 资源建筑最高10级
```

## 四、资源产出

### 4.1 基础产出

```
每级产出 = 100 × 等级

示例:
民居5级: 500金币/小时
灵田5级: 250灵气/小时
```

### 4.2 产出公式

```
每小时产出 = 基础产出 × (1 + 加成)

加成来源:
- 联盟科技: 最高+30%
- 城池科技: 最高+20%
- 道具: 最高+50%
```

## 六、迁城系统

### 6.1 迁城条件

```
等级要求: 5级
CD时间: 7天
消耗资源: 1000灵气
```

### 6.2 迁城限制

```
只能迁到同区域:
- 出生州 → 出生州
- 资源州 → 资源州
- 朝歌 → 朝歌
```

### 6.3 迁城惩罚

```
迁城后:
- 建筑等级保留
- 资源清零
- CD开始计时
```
