## 三、兵种系统

### 3.1 兵种分类

| 兵种 | 伤害类型 | 特点 |
|------|----------|------|
| 步(步兵) | 物理 | 高防御 |
| 骑(骑兵) | 物理 | 高攻击 |
| 弓(弓兵) | 物理 | 高速度 |
| 策(策士) | 法术 | 高智力 |

### 3.2 兵种克制

```
骑 → 步 (攻击+20%)
步 → 弓 (攻击+20%)
弓 → 骑 (攻击+20%)
策 无克制 (法术伤害)
```

## 四、等级系统

### 4.1 等级上限

```
所有武将统一最高100级
```

### 4.2 升级经验

```
升级经验 = 等级 × 150

示例:
1→2级: 150经验
10→11级: 1500经验
50→51级: 7500经验
```

### 4.3 成长值

| 品质 | 武力/级 | 智力/级 | 统率/级 | 速度/级 |
|------|---------|---------|---------|---------|
| 红 | 2.0 | 2.0 | 1.5 | 1.0 |
| 橙 | 1.6 | 1.6 | 1.2 | 0.8 |
| 紫 | 1.2 | 1.2 | 0.9 | 0.6 |
| 蓝 | 0.8 | 0.8 | 0.6 | 0.4 |

### 4.4 属性公式

```
当前属性 = 基础属性 + (等级 - 1) × 成长值
```

### 4.5 100级属性示例

**红将哪吒** (基础武力160, 成长2.0):
```
100级武力 = 160 + 99×2.0 = 358
```

**橙将金吒** (基础武力200, 成长1.6):
```
100级武力 = 200 + 99×1.6 = 358
```

## 五、星级系统

### 5.1 星级加成

| 星级 | 属性加成 | 升级消耗 |
|------|----------|----------|
| 1星 | +0% | - |
| 2星 | +15% | 2个相同 |
| 3星 | +30% | 2个相同 |
| 4星 | +50% | 2个相同 |
| 5星 | +75% | 2个相同 |
| 6星 | +100% | 2个相同 |

### 5.2 升星消耗

```
升星公式: 2个相同武将 = 1星

2星: 2个
3星: 4个 (累计)
4星: 6个 (累计)
5星: 8个 (累计)
6星: 10个 (累计)

满星(6星)需要10个相同武将
```

### 5.3 星级属性公式

```
实际属性 = 基础属性 × (1 + 星级加成)
```

### 5.4 星级战力示例

**哪吒 100级**:
- 1星武力: 358 × 1.0 = 358
- 3星武力: 358 × 1.3 = 465
- 6星武力: 358 × 2.0 = 716

## 七、战力计算

### 7.1 战力公式

```
战力 = (武力 + 智力 + 统率) × 2 + 速度 + 兵量/100 + 技能加成

技能加成 = 技能数 × 50 + 技能效果值
```

### 7.2 战力示例

**哪吒 100级 6星**:
```
武力: 716
智力: 446
统率: 358
速度: 289
兵量: 60000

基础战力: (716+446+358)×2 + 289 = 3040
兵量加成: 60000/100 = 600
技能加成: 2×50 + 105 = 205

总战力: 3040 + 600 + 205 = 3845
```
