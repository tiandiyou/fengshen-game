## 六、锻造系统

### 6.1 锻造流程

```
1. 选择装备类型 (武器/防具/头盔/鞋子/饰品)
2. 选择材料
3. 点击锻造
4. 随机生成装备品质
5. 随机生成特技 (概率触发)
6. 获得装备
```

### 6.2 锻造概率

```
蓝色装备: 60%
紫色装备: 28%
橙色装备: 10%
红色装备: 2%

特技触发:
- 蓝色: 5%
- 紫色: 10%
- 橙色: 20%
- 红色: 30%
```

### 6.3 强化流程

```
1. 选择装备
2. 选择材料
3. 点击强化
4. 成功率100% (设计简化)
5. 属性提升
```

## 七、装备数据结构

### 7.1 装备实体

```java
class Equipment {
    Long id;              // 装备ID
    String name;          // 装备名称
    String type;          // 类型: weapon/armor/helmet/shoes/accessory
    String quality;       // 品质: red/orange/purple/blue/green
    
    // 基础属性
    Integer atk;          // 攻击
    Integer def;          // 防御
    Integer hp;           // 生命
    Integer speed;        // 速度
    Integer crit;         // 暴击
    
    // 强化
    Integer enhanceLevel; // 强化等级
    
    // 特技
    String skill;         // 特技名称
    String skillLevel;    // 特技等级: S/A/B/C
    
    // 专属
    Integer partnerId;    // 专属武将ID (0=非专属)
    
    // 状态
    Boolean equipped;     // 是否已装备
    Long ownerId;        // 拥有者玩家ID
}
```

### 7.2 材料实体

```java
class Material {

    String name;          // 材料名称
    String type;          // 材料类型
    Integer count;        // 数量

}
```

## 八、战斗系统整合

### 8.1 属性计算

```
最终武力 = 武将武力 + 装备武力 × (1 + 强化等级 × 10%)
最终智力 = 武将智力 + 装备智力 × (1 + 强化等级 × 10%)
最终防御 = 武将防御 + 装备防御 × (1 + 强化等级 × 10%)
最终速度 = 武将速度 + 装备速度 × (1 + 强化等级 × 10%)
```

### 8.2 特技触发

```
战斗回合开始时:
  检查所有已装备装备
  触发被动特技 (如铁壁、疾风)
  
伤害计算时:
  触发伤害类特技 (如猛攻、破甲)
  
受到伤害时:
  触发防御类特技 (如铁壁、反击)
```
