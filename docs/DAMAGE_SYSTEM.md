## 一、物理伤害

### 1.1 伤害公式

```
物理伤害 = 武力 × (1 - 统率 / (统率 + 200))
```

### 1.2 公式解析

- **攻击力**: 武将的武力值
- **防御力**: 敌方的统率值
- **穿透系数**: 200为防御基准值

### 1.3 伤害示例表

| 武力 | 统率 | 伤害 | 减伤率 |
|------|------|------|--------|
| 180 | 0 | 180 | 0% |
| 180 | 50 | 144 | 20% |
| 180 | 100 | 120 | 33% |
| 180 | 150 | 103 | 43% |
| 180 | 180 | 95 | 47% |
| 180 | 200 | 90 | 50% |
| 180 | 300 | 72 | 60% |
| 180 | 500 | 52 | 71% |
| 180 | 1000 | 35 | 81% |

### 1.4 最低伤害保障

```
最低伤害 = 1
```

即使防御极高，攻击方至少造成1点伤害，保证战斗必将继续。

## 二、法术伤害

### 2.1 伤害公式

```
法术伤害 = 智力 × (1 - 敌方智力 / (敌方智力 + 200))
```

### 2.2 公式解析

- **攻击力**: 武将的智力值
- **防御力**: 敌方的智力值
- **穿透系数**: 200为防御基准值

### 2.3 伤害示例表

| 智力 | 敌方智力 | 伤害 | 减伤率 |
|------|----------|------|--------|
| 200 | 0 | 200 | 0% |
| 200 | 50 | 160 | 20% |
| 200 | 100 | 133 | 33% |
| 200 | 150 | 114 | 43% |
| 200 | 200 | 100 | 50% |
| 200 | 300 | 80 | 60% |
| 200 | 500 | 57 | 71% |

## 三、伤害类型判断

### 3.1 兵种伤害类型

| 兵种 | 伤害类型 |
|------|----------|
| 步(步兵) | 物理伤害 |
| 骑(骑兵) | 物理伤害 |
| 弓(弓兵) | 物理伤害 |
| 策(策士) | 法术伤害 |

### 3.2 技能伤害类型

技能伤害类型取决于技能本身：
- 火尖枪 → 物理伤害
- 封神榜 → 法术伤害

## 四、技能伤害加成

### 4.1 物理伤害技能

```
技能伤害 = 物理伤害 × (1 + 技能加成)

示例:
- 火尖枪(55%): 伤害 = 物理伤害 × 1.55
- 八九玄功(50%): 伤害 = 物理伤害 × 1.50
```

### 4.2 法术伤害技能

```
技能伤害 = 法术伤害 × (1 + 技能加成)

示例:
- 封神榜(55%): 伤害 = 法术伤害 × 1.55
- 开天珠(50%): 伤害 = 法术伤害 × 1.50
```

## 五、暴击伤害

### 5.1 暴击公式

```
暴击伤害 = 基础伤害 × 暴击倍率
暴击倍率默认 = 1.5
```

### 5.2 暴击触发

```
暴击概率 = 暴击率 - 敌方抗暴率
```

---

## 六、综合伤害计算

### 6.1 完整公式

```
最终伤害 = 基础伤害 × (1 + 技能加成) × (1 + 暴击加成) × 随机浮动

随机浮动范围: 0.9 ~ 1.1
```

### 6.2 计算示例

**哪吒使用火尖枪攻击闻仲**:

```
哪吒武力: 180
闻仲统率: 150

基础物理伤害 = 180 × (1 - 150/350) = 180 × 0.571 = 103

火尖枪技能加成 +55%
技能伤害 = 103 × 1.55 = 160

暴击(假设触发, 1.5倍)
最终伤害 = 160 × 1.5 × 1.0(随机) = 240
```

## 七、伤害公式对比

### 7.1 方案A (减法)

```
伤害 = 攻击 - 防御
问题: 防御 > 攻击时伤害为0
```

### 7.2 方案B (百分比穿透) ✓

```
伤害 = 攻击 × (1 - 防御 / (防御 + 200))
优势: 永远有伤害，双方属性都有意义
```

## 八、代码实现

```java
/**
 * 物理伤害计算
 */
public int calcPhysicalDamage(int atk, int enemyLead) {
    double dmg = atk * (1 - (double)enemyLead / (enemyLead + 200));
    return Math.max(1, (int) dmg);
}

/**
 * 法术伤害计算
 */
public int calcMagicDamage(int intVal, int enemyInt) {
    double dmg = intVal * (1 - (double)enemyInt / (enemyInt + 200));
    return Math.max(1, (int) dmg);
}
```
