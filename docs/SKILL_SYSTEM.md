## 一、技能分类

### 1.1 技能类型总览

| 类型 | 效果 | 战斗中使用 | 占比 |
|------|------|------------|------|
| 伤害类 | 物理/法术伤害加成 | 主动 | 30% |
| 控制类 | 石化/沉默/眩晕 | 主动 | 25% |
| 增益类 | 攻击+/防御+/速度+ | 被动/主动 | 25% |
| 恢复类 | 治疗/护盾 | 主动 | 15% |
| 特殊类 | 暴击/闪避/反击 | 被动 | 5% |

### 1.2 技能释放时机

| 类型 | 释放时机 |
|------|----------|
| 主动技能 | 玩家手动释放 |
| 被动技能 | 战斗常驻生效 |
| 指挥战法 | 战斗开始时生效 |
| 联动技能 | 特定组合触发 |

## 四、技能数据结构

### 4.1 技能实体

```java
class Skill {
    int id;              // 技能ID
    String name;         // 技能名称
    String type;        // 类型: damage/control/buff/heal/special
    String damageType;   // 伤害类型: physical/magic
    int effectValue;     // 效果值
    String effect;       // 效果描述
    String quality;      // 品质: red/orange/purple/blue
    boolean isActive;    // 是否主动技能
    int turns;           // 持续回合数(控制类)
}
```

### 4.2 技能效果示例

```json
{
    "id": 101,
    "name": "火尖枪",
    "type": "damage",
    "damageType": "physical",
    "effectValue": 55,
    "effect": "物理伤害+55%",
    "quality": "red",
    "isActive": true,
    "turns": 0
}
```

## 五、技能触发

### 5.1 主动技能触发

```
战斗回合中，玩家选择释放主动技能
↓
技能效果立即生效
↓
进入下一回合
```

### 5.2 被动技能触发

```
战斗开始时
↓
被动技能自动生效
↓
全程享受增益
```

### 5.3 指挥战法触发

```
己方全体武将上阵时
↓
指挥战法效果覆盖全体
↓
持续整场战斗
```

## 七、技能升级

### 7.1 升级消耗

| 技能等级 | 消耗灵气 |
|----------|----------|
| 1→2 | 100 |
| 2→3 | 200 |
| 3→4 | 400 |
| 4→5 | 800 |
| 5→6 | 1600 |

### 7.2 升级效果

```
技能效果 = 基础效果 × (1 + 等级 × 10%)

示例:
火尖枪1级: 物理+55%
火尖枪5级: 物理+55% × 1.5 = 82.5%
```
