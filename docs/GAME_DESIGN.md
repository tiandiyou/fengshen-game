# 封神榜 - 整体游戏设计文档

**版本**: v1.0  
**更新时间**: 2026-03-15  
**状态**: 核心框架

---

## 一、游戏定位

### 1.1 类型
```
SLG (策略游戏) + 卡牌养成
```

### 1.2 核心玩法
```
配将 + 配技能 + 联动效果 → 攻打朝歌
```

---

## 二、赛季系统

### 2.1 赛季目标
```
最终目标: 攻打朝歌 (地图中心)
```

### 2.2 赛季流程
```
1. 赛季开始
2. 出生在低等级地图
3. 发育资源
4. 加入/创建联盟
5. 联盟组队攻打关卡/城池
6. 进入资源州 (资源丰富)
7. 继续发育
8. 进攻朝歌
9. 赛季结束
```

### 2.3 跨赛季继承

| 内容 | 跨赛季保留 | 跨赛季重置 |
|------|------------|------------|
| 武将 | ✅ | - |
| 技能 | ✅ | - |
| 修仙等级 | - | ✅ |
| 装备 | - | ✅ |
| 材料 | - | ✅ |

---

## 三、大地图系统

### 3.1 地图结构

```
┌────────────────────────────────────────┐
│              资源州 (资源丰富)           │
│    ┌────────────────────────────────┐  │
│    │          资源州                  │  │
│    │  ┌──────────────────────────┐  │  │
│    │  │        资源州             │  │  │
│    │  │    ┌────────────────┐   │  │  │
│    │  │    │     朝歌        │   │  │  │
│    │  │    │   (最终目标)    │   │  │  │
│    │  │    └────────────────┘   │  │  │
│    │  └──────────────────────────┘  │  │
│    └────────────────────────────────┘  │
│                                         │
│  ┌───────────┐    ┌───────────┐        │
│  │ 出生州1  │    │ 出生州2   │        │
│  │ (低级)   │    │  (低级)  │        │
│  └───────────┘    └───────────┘        │
└────────────────────────────────────────┘
```

### 3.2 地图层级

| 层级 | 名称 | 资源 | 难度 |
|------|------|------|------|
| 1级 | 出生州 | 低 | 低 |
| 2级 | 资源州 | 高 | 中 |
| 3级 | 朝歌 | 最高 | 高 |

### 3.3 行军遇怪

```
玩家在大地图上行军
↓
随机触发野将战斗
↓
战斗胜利 → 材料掉落
战斗失败 → 少量材料/无掉落
↓
继续行军
```

## 四、联盟系统

### 4.1 联盟功能

| 功能 | 说明 |
|------|------|
| 组队战斗 | 联盟成员组队攻打关卡 |
| 城池占领 | 联盟占领城池获得资源 |
| 联盟聊天 | 成员交流 |
| 联盟仓库 | 共用资源 |

### 4.2 城池系统

```
联盟攻打城池
↓
胜利后联盟占领
↓
联盟成员获得:
- 每日资源产出
- 城池经验加成
- 专属道具
```

### 4.3 关卡

```
关卡 = 通往资源州的通道
↓
需要联盟共同攻克
↓
攻克后联盟成员可进入
```

## 五、武将系统

### 5.1 核心特点

- **配将**: 根据武将特点组合阵容
- **配技能**: 技能搭配产生联动
- **联动效果**: 特定组合触发额外加成

### 5.2 跨赛季

```
武将: 跨赛季保留
技能: 跨赛季保留
星级: 跨赛季保留
等级: 跨赛季继承
```

---

## 六、技能系统

### 6.1 技能分类

| 类型 | 效果 |
|------|------|
| 主动技能 | 手动释放 |
| 被动技能 | 常驻生效 |
| 指挥战法 | 战斗开始生效 |
| 联动技能 | 组合触发 |

### 6.2 联动示例

| 组合 | 武将 | 激活效果 |
|------|------|----------|
| 阐教弟子 | 姜子牙+杨戬+雷震子 | 全体攻击+15% |
| 截教弟子 | 申公豹+赵公明+魔礼青 | 全体防御+15% |
| 皇室血脉 | 纣王+伯邑考+周武王 | 全体生命+20% |

### 6.3 跨赛季

```
技能: 跨赛季保留
技能等级: 跨赛季保留
```

## 七、修仙系统

### 7.1 修仙说明

```
任意武将可以修炼
↓
修炼增加武将属性
↓
跨赛季重置
```

### 7.2 修炼属性

| 修炼方向 | 增加属性 |
|----------|----------|
| 炼体 | 武力+5/级 |
| 炼神 | 智力+5/级 |
| 炼气 | 统率+3/级 |
| 炼虚 | 速度+3/级 |

### 7.3 修炼消耗

```
每次修炼: 灵气 × 100

修仙等级上限: 50级
```

### 7.4 跨赛季重置

```
赛季结束
↓
修仙等级重置为1
↓
修仙材料清空
↓
武将属性回归基础
```

## 十、战斗系统

### 10.1 战斗模式

```
自动战斗 + 手动技能释放
```

### 10.2 战斗流程

```
1. 双方布阵 (6将)
2. 回合制战斗
3. 玩家手动释放主动技能
4. 被动技能常驻
5. 指挥战法开场生效
6. 联动技能组合触发
7. 兵量归零判定胜负
```

---

## 十一、成长循环

### 11.1 核心循环

```
抽卡/培养武将 → 配置技能 → 组建阵容
    ↓
大地图行军 → 打怪获材料 → 合成装备
    ↓
提升实力 → 联盟组队 → 攻打城池
    ↓
进入资源州 → 发育资源 → 进攻朝歌
    ↓
赛季结束 → 武将技能保留 → 重新开始
```

### 11.2 跨赛季节奏

```
每个赛季:
1. 用上赛季武将+技能
2. 重新发育资源
3. 重新打材料做装备 (装备不保留)
4. 重新修仙 (修仙重置)
5. 冲击朝歌
```
