## 一、联盟系统

### 1.1 联盟基础

```
联盟 = 玩家组织
玩家可以创建/加入联盟
联盟成员共同战斗
```

### 1.2 创建联盟

| 条件 | 消耗 |
|------|------|
| 等级要求 | 10级 |
| 创建消耗 | 500元宝 |
| 联盟名称 | 2-10字 |
| 联盟标志 | 选择图标 |

### 1.3 联盟人数

| 联盟等级 | 人数上限 |
|----------|----------|
| 1级 | 20人 |
| 2级 | 30人 |
| 3级 | 40人 |
| 4级 | 50人 |
| 5级 | 100人 |

### 1.4 联盟升级

```
升级消耗: 联盟贡献
升级条件: 成员数量达到要求
```

## 三、联盟功能

### 3.1 联盟商店

```
联盟成员贡献可兑换:
- 稀有武将碎片
- 进阶材料
- 装备材料
- 道具
```

### 3.2 联盟仓库

```
成员可存放材料
其他成员可免费取用
```

### 3.3 联盟科技

```
联盟共同研发科技
所有成员获得加成

| 科技 | 效果 | 消耗 |
|------|------|------|
| 攻击强化 | 全体成员攻击+5% | 1000贡献 |
| 防御强化 | 全体成员防御+5% | 1000贡献 |
| 资源增产 | 全体成员资源+10% | 1500贡献 |
```

### 3.4 联盟任务

```
每日任务:
- 贡献资源
- 参加联盟战
- 助攻盟友

奖励: 联盟贡献 + 道具
```

## 四、战队系统

### 4.1 战队说明

```
联盟内可创建战队
战队 = 小队 (3-5人)
```

### 4.2 战队创建

| 条件 | 消耗 |
|------|------|
| 盟主/副盟主 | 免费创建 |
| 人数 | 3-5人 |
| 名称 | 2-8字 |

### 4.3 战队功能

```
1. 组队战斗
   - 战队成员可组队打关卡
   - 共享经验加成

2. 集结进攻
   - 战队可集结进攻城池
   - 成员同时参战

3. 战队聊天
   - 战队内部频道
```

### 4.4 战队职位

| 职位 | 人数 | 权限 |
|------|------|------|
| 队长 | 1人 | 管理/踢人 |
| 副队长 | 1人 | 邀请成员 |
| 成员 | 3人 | 参与战斗 |

## 五、好友系统

### 5.1 好友功能

```
好友 = 玩家列表
可查看好友在线状态
可发送消息
可互赠体力
```

### 5.2 好友数量

| 玩家等级 | 好友上限 |
|----------|----------|
| 1-10级 | 20人 |
| 11-20级 | 30人 |
| 21-30级 | 50人 |
| 31级以上 | 100人 |

### 5.3 好友互动

| 互动 | 次数 | 奖励 |
|------|------|------|
| 赠送体力 | 3次/天 | 双方+5体力 |
| 点赞 | 无限 | 友好度+1 |
| 聊天 | 无限 | - |

### 5.4 友好度

```
友好度作用:
- 组队战斗加成
- 互赠体力上限

| 友好度 | 等级 |
|--------|------|
| 0-99 | 1级 |
| 100-499 | 2级 |
| 500-999 | 3级 |
| 1000+ | 4级 |
```

## 六、联盟战争

### 6.1 城池争夺

```
联盟可攻打城池
胜利后联盟占领
成员获得:
- 每日资源产出
- 专属道具
```

### 6.2 宣战系统

```
宣战消耗: 1000元宝
宣战后:
- 48小时内可进攻
- 防守方应战
- 胜利方获得城池
```

### 6.3 关卡攻略

```
联盟共同攻略关卡:
- 成员分别进攻
- 累计伤害达标即可通关
- 通关后进入资源州
```

## 七、数据结构

### 7.1 联盟

```java
class Alliance {
    Long id;
    String name;
    String icon;
    Long leaderId;       // 盟主ID
    
    Integer level;       // 联盟等级
    Integer memberCount; // 成员数
    
    String notice;      // 公告
    
    List<AllianceMember> members;
}
```

### 7.2 联盟成员

```java
class AllianceMember {
    Long id;
    Long playerId;
    Long allianceId;
    
    String position;     // 盟主/副盟主/长老/精英/成员
    
    Integer contribution; // 累计贡献
    Integer todayContrib; // 今日贡献
}
```

### 7.3 战队

```java
class Team {
    Long id;
    Long allianceId;
    
    String name;
    Long leaderId;     // 队长
    
    List<TeamMember> members;  // 3-5人
}
```

### 7.4 好友

```java
class Friend {
    Long id;
    Long playerId;
    Long friendId;
    
    Integer friendliness; // 友好度
    
    LocalDateTime lastGiftTime; // 上次送礼时间
}
```
