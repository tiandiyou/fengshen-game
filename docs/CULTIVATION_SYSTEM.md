# 修仙系统设计文档

**版本**: v1.0  
**更新**: 2026-03-15  
**定位**: SLG赛季制

---

## 一、核心设计

### 1.1 修仙说明

```
玩家进行修仙修炼
↓
修仙产出"灵气结晶"
↓
消耗灵气结晶给武将修炼
↓
武将属性提升
↓
跨赛季重置
```

### 1.2 修炼特点

| 特点 | 说明 |
|------|------|
| 幅度小 | 每次提升少量属性 |
| 上限高 | 最高可提升至红将水平 |
| 需要时间 | 修炼需要等待 |
| 跨赛季 | 重置为1级 |

---

## 二、修仙系统

### 2.1 修仙等级

```
修仙等级: 1-50级
每升1级增加产出
```

### 2.2 升级消耗

| 等级 | 升级经验 | 所需时间 |
|------|----------|----------|
| 1→2 | 100 | 1分钟 |
| 2→3 | 200 | 2分钟 |
| ... | ... | ... |
| 49→50 | 4900 | 49分钟 |

**公式**: `升级时间(分钟) = 当前等级`

### 2.3 产出公式

```
每小时产出 = 修仙等级 × 10

示例:
10级修仙 = 100个灵气结晶/小时
50级修仙 = 500个灵气结晶/小时
```

---

## 三、修炼系统

### 3.1 修炼消耗

```
每次修炼消耗: 50个灵气结晶
修炼间隔: 10分钟
```

### 3.2 属性提升

| 方向 | 增加属性 |
|------|----------|
| 炼体 | 武力+2 |
| 炼神 | 智力+2 |
| 炼气 | 统率+2 |
| 炼虚 | 速度+1 |

### 3.3 修炼上限

```
单个属性修炼上限 = 橙将基础属性 - 武将基础属性

示例:
金吒(橙将):
- 武力基础: 130
- 金吒武力: 130
- 修炼上限: 0 (无法修炼)

哪吒(红将):
- 红将武力基准: 180
- 哪吒武力: 160
- 修炼上限: 20 (可提升20点武力)
```

### 3.4 上限参考表

| 武将 | 属性 | 基础 | 红将基准 | 可提升 |
|------|------|------|----------|--------|
| 哪吒 | 武力 | 160 | 180 | +20 |
| 哪吒 | 智力 | 75 | 120 | +45 |
| 哪吒 | 统率 | 80 | 120 | +40 |
| 哪吒 | 速度 | 90 | 130 | +40 |
| 杨戬 | 武力 | 150 | 180 | +30 |
| 金吒 | 武力 | 130 | 180 | +50 |

## 四、修炼流程

### 4.1 修炼步骤

```
1. 选择武将
2. 选择修炼方向 (炼体/炼神/炼气/炼虚)
3. 点击修炼
4. 消耗50灵气结晶
5. 等待10分钟
6. 属性自动提升
```

### 4.2 修炼加速

```
消耗元宝立即完成修炼
价格: 10元宝/次
```

## 五、数据结构

### 5.1 修仙数据

```java
class Cultivation {
    Long id;
    Long playerId;
    
    Integer level = 1;        // 修仙等级
    Integer exp = 0;         // 修仙经验
    Integer totalExp;         // 累计经验
    LocalDateTime lastUpdate; // 上次产出时间
    
    // 产出
    Integer getHourlyOutput() {
        return level * 10;
    }
}
```

### 5.2 修炼数据

```java
class Practice {
    Long id;
    Long playerId;
    Long partnerId;          // 武将ID
    
    Integer trainAtk = 0;    // 炼体累计
    Integer trainInt = 0;    // 炼神累计
    Integer trainLead = 0;   // 炼气累计
    Integer trainSpeed = 0;  // 炼虚累计
    
    Integer maxAtk;          // 武力上限
    Integer maxInt;          // 智力上限
    Integer maxLead;         // 统率上限
    Integer maxSpeed;        // 速度上限
    
    LocalDateTime lastPractice; // 上次修炼时间
}
```

## 六、跨赛季规则

```
赛季结束:
- 修仙等级: ✅ 保留
- 武将修炼: ❌ 重置
- 灵气结晶: ❌ 保留 (不清空)

新赛季:
- 武将修炼属性归零
- 重新消耗灵气结晶修炼
```

## 九、总结

### 核心设计

| 内容 | 说明 |
|------|------|
| 修仙等级 | 1-50级 |
| 修仙产出 | 等级×10个灵气结晶/小时 |
| 修炼消耗 | 50个/次 |
| 修炼间隔 | 10分钟/次 |
| 属性提升 | +2/次 (速度+1) |
| 跨赛季 | 修炼重置，结晶保留 |

### 提升幅度

```
50级修仙 + 持续修炼 ≈ 可将蓝将提升至接近红将水平

示例:
蓝将武力: 80
持续修炼50次(约8小时): +100武力
最终武力: 180 ≈ 红将水平
```
