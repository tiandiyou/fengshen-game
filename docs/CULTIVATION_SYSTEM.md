## 二、修仙分支

### 2.1 两大流派

```
体修 (物理系) → 增加武将属性
法修 (法术系) → 增加技能伤害
```

### 2.2 选择

```
创角时选择流派
流派不可更改
```

## 三、升级时间 (免费)

### 3.1 设计依据

```
一赛季: 2个月 (60天)
满级: 50级
免费速度: 1级/天
```

### 3.2 升级时间表

| 等级 | 累计时间 | 完成时间 |
|------|----------|----------|
| 1→2 | 1天 | 第1天 |
| 2→3 | 2天 | 第2天 |
| ... | ... | ... |
| 25→26 | 25天 | 第25天 |
| 49→50 | 50天 | 第50天 |

**免费玩家满级时间: 约50天 ≈ 2个月**

---

## 四、结晶加速

### 4.1 加速公式

```
每加速1天 = 消耗100结晶

公式: 加速天数 × 100 = 消耗结晶
```

### 4.2 加速示例

```
从1级升到50级:
- 纯免费: 50天
- 加速30天: 消耗3000结晶, 20天完成
- 瞬间满级: 消耗4900结晶 (约50天)
```

### 4.3 结晶消耗表

| 目标等级 | 免费时间 | 加速消耗 |
|----------|----------|----------|
| 10级 | 10天 | 0结晶 |
| 20级 | 20天 | 0结晶 |
| 30级 | 30天 | 0结晶 |
| 40级 | 40天 | 0结晶 |
| 50级 | 50天 | 0结晶 |

**免费玩家**: 2个月可满级  
**付费玩家**: 可用结晶加速

```
结晶: 赛季商城专属货币

```

### 5.2 商城价格

| 数量 | 价格 | 性价比 |
|------|------|----------|
| 100个 | 100元宝 | 1元宝=1结晶 |
| 500个 | 450元宝 | 9折 |

```
普通玩家: 500个 (450元宝) = 加速5天
```

## 七、数据结构

### 7.1 修仙数据

```java
class Cultivation {
    Long id;
    Long playerId;
    
    String type;  // "body"=体修, "magic"=法修
    
    Integer level = 1;    // 当前等级
    Integer exp = 0;      // 当前进度(天)
    
    LocalDateTime lastUpdate; // 上次更新时间
    
    // 升级所需
    int getDaysToNextLevel() {
        return level;  // 1级需要1天, 2级需要2天...
    }
    
    // 结晶加速
    int getAccelerateCost(int days) {
        return days * 100;  // 1天=100结晶
    }
}
```

### 7.2 赛季数据

```java
class SeasonPlayerData {
    Long playerId;
    Integer seasonId;
    
    // 修仙 (赛季重置)
    Integer cultivationLevel;
    
    // 结晶 (跨赛季保留)
    Integer crystalCount;
}
```
