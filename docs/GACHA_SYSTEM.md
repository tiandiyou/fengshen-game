## 一、5+1模式

### 1.1 模式说明

```
玩家选择5个武将 + 系统随机1个武将 = 6人阵容
```

### 1.2 阵容构成

| 位置 | 选择方式 |
|------|----------|
| 位置1-5 | 玩家自主选择 |
| 位置6 | 系统随机 |

## 二、保底机制

### 2.1 保底规则

```
┌────────────────────────────────────────┐
│  20抽 → 必出1个橙色及以上武将           │
│  累计6个橙色武将 → 第6个必定是红色武将   │
└────────────────────────────────────────┘
```

### 2.2 保底计数器

```java
class GachaRecord {
    int totalDraws;      // 总抽卡次数
    int orangeCount;    // 累计橙将数量
    int pityCounter;    // 保底计数器 (满20清零)
}
```

### 2.3 保底流程

```
第1-19抽:
  ↓
判断是否触发20抽保底
  ↓
第20抽:
  必出橙将 (保底)
  orangeCount + 1
  pityCounter = 0
  ↓
判断累计6橙:
  是 → 下一个必出红将
  否 → 继续普通抽取
```

## 三、基础概率

### 3.1 概率表

| 品质 | 颜色 | 概率 |
|------|------|------|
| 红 | 红色 | 1% |
| 橙 | 橙色 | 5% |
| 紫 | 紫色 | 24% |
| 蓝 | 蓝色 | 70% |

### 3.2 概率公式

```
随机数 = Math.random() × 100

if (随机数 < 1)       → 红将
else if (随机数 < 6)  → 橙将 (1+5)
else if (随机数 < 30)  → 紫将 (6+24)
else                   → 蓝将
```

## 四、名将加成

### 4.1 加成说明

```
名将效果只影响抽卡概率，不影响其他系统
```

### 4.2 加成表

| 名将类型 | 加成值 | 拥有后红将概率 |
|----------|--------|----------------|
| 无 | 0% | 1% |
| 良将 | +1% | 2% |
| 名将 | +3% | 4% |
| 神将 | +5% | 6% |

### 4.3 加成计算

```
实际红将概率 = 基础概率 + 名将加成

示例:
玩家拥有纣王(神将+5%)
红将概率 = 1% + 5% = 6%
```

### 4.4 多名将叠加

```
名将加成取最高值，不叠加

示例:
同时拥有神将(+5%)和名将(+3%)
实际加成 = 5% (取最高)
```

## 五、抽卡流程

### 5.1 单抽流程

```
1. 玩家点击单抽
2. 检查20抽保底:
   - 是 → 必出橙将
   - 否 → 进入概率判定
3. 检查6橙保底:
   - 是 → 必出红将
   - 否 → 普通概率
4. 随机抽取武将
5. 记录抽卡结果
6. 返回武将
```

### 5.2 十连抽流程

```
1. 玩家点击十连抽
2. 循环10次单抽流程
3. 必定获得至少1个橙将
4. 返回10个武将
```

### 5.3 5+1抽流程

```
1. 玩家选择5个武将
2. 系统随机1个武将
3. 组合6人阵容
4. 计入抽卡次数
```

## 八、数据结构

### 8.1 抽卡记录实体

```java
@Entity
class GachaRecord {
    @Id
    Long id;
    
    Long playerId;       // 玩家ID
    
    Integer gachaType;   // 抽卡类型: 1=单抽, 10=十连, 51=5+1
    
    Integer totalDraws;   // 总抽卡次数

    Integer pityCounter; // 保底计数器
    
    String result;       // 抽卡结果 (JSON)
    
    LocalDateTime drawTime; // 抽卡时间
}
```

    int blueCount;           // 蓝将数量

## 九、代码实现

### 9.1 抽卡服务

```java
@Service
class GachaService {
    
    // 基础概率
    private static final double RED_RATE = 0.01;
    private static final double ORANGE_RATE = 0.05;
    private static final double PURPLE_RATE = 0.24;
    
    // 保底阈值
    private static final int PITY_THRESHOLD = 20;
    private static final int ORANGE_PITY = 6;
    
    public GachaResult draw(GachaRecord record, int gachaType) {
        // 1. 检查20抽保底
        if (record.getPityCounter() >= PITY_THRESHOLD - 1) {
            return drawOrangePity(record);
        }
        
        // 2. 检查6橙保底
        if (record.getOrangeCount() >= ORANGE_PITY - 1) {
            return drawRedPity(record);
        }
        
        // 3. 普通概率抽取
        return drawNormal(record);
    }
}
```
