## 二、行军系统

### 2.1 行军规则

```
玩家在地图上移动
↓
选择目标点
↓
队伍行军前往
↓
到达后触发事件
```

### 2.2 行军速度

```
基础速度: 100/分钟
速度加成:
- 骑兵: +20%
- 坐骑速度属性: +10/点

计算公式: 实际速度 = 基础速度 × (1 + 加成)
```

### 2.3 遇怪触发

```
行军途中随机触发战斗
↓
触发概率: 30%
↓
触发后:
- 战斗胜利: 获得材料
- 战斗失败: 少量材料/无奖励
↓
继续行军
```

### 2.4 材料掉落

| 区域 | 玄铁 | 精钢 | 陨金 | 天晶 |
|------|------|------|------|------|
| 出生州 | 80% | 20% | - | - |
| 资源州 | 30% | 40% | 25% | 5% |
| 朝歌 | 10% | 25% | 45% | 20% |

## 三、自动挂机

### 3.1 挂机功能

```
玩家可设置队伍自动巡逻
↓
队伍自动在区域内移动
↓
移动时自动触发遇怪战斗
↓
战斗自动进行
↓
自动获得材料
```

### 3.2 挂机设置

```
挂机区域: 只能在当前所在区域
挂机范围: 可选择全图/指定点
挂机时间: 可设置1小时/3小时/无限
```

### 3.3 挂机收益

| 时间 | 战斗次数 | 预计收益 |
|------|----------|----------|
| 1小时 | 3-5次 | 10-20个材料 |
| 3小时 | 10-15次 | 30-50个材料 |
| 8小时 | 25-35次 | 80-120个材料 |

### 3.4 注意事项

```
挂机时:
- 队伍需要有足够兵量
- 兵量不足自动停止
- 可设置最低兵量阈值
```

## 四、关卡系统

### 4.1 关卡定位

```
关卡 = 进入下一区域的通道
↓
需要攻克关卡才能进入
```

### 4.2 关卡分布

| 关卡 | 位置 | 解锁条件 |
|------|------|----------|
| 出生州→资源州 | 出生州边界 | 通关6章战斗 |
| 资源州→朝歌 | 资源州边界 | 联盟攻克 |

### 4.2 关卡守将

| 关卡 | 守将 | 等级 | 难度 |
|------|------|------|------|
| 出生州→资源州 | 魔礼青 | 30级 | 中等 |
| 资源州→朝歌 | 闻仲 | 50级 | 困难 |

### 4.3 攻克规则

```
需要联盟成员共同攻打
↓
个人可单独进攻
↓
累计伤害达标即通关
↓
联盟成员可进入下一区域
```

## 五、城池系统

### 5.1 城池定位

```
城池 = 联盟占领的建筑
↓
占领后获得资源产出
```

### 5.2 城池类型

| 城池 | 等级 | 产出 | 解锁 |
|------|------|------|------|
| 县城 | 1级 | 金币+10% | 出生州 |
| 郡城 | 2级 | 粮草+10% | 资源州 |
| 州城 | 3级 | 木材+10% | 资源州 |
| 都城 | 4级 | 灵气+10% | 朝歌 |

### 5.3 城池守将

```
城池有NPC守将
守将等级根据城池等级
击败守将即可占领
```

### 5.4 占领规则

```
1. 联盟宣战
2. 48小时内进攻
3. 击败守将
4. 联盟占领
5. 成员获得产出加成
```

### 5.5 城池产出

| 城池 | 等级 | 每日产出 |
|------|------|----------|
| 县城 | 1级 | 1000金币 |
| 郡城 | 2级 | 1000粮草 |
| 州城 | 3级 | 1000木材 |
| 都城 | 4级 | 100灵气 |

## 六、遗迹系统

### 6.1 遗迹分布

| 区域 | 遗迹 | 难度 |
|------|------|------|
| 出生州 | 山洞 | 简单 |
| 出生州 | 古墓 | 中等 |
| 资源州 | 仙府 | 困难 |

### 6.2 遗迹详情

| 遗迹 | 挑战次数 | 品质 | 红坐骑 |
|------|----------|------|--------|
| 山洞 | 1次/天 | 蓝/紫 | 0% |
| 古墓 | 1次/天 | 紫/橙 | 2% |
| 仙府 | 1次/天 | 橙/红 | 10% |

### 6.3 挑战规则

```
点击遗迹
↓
选择队伍进入
↓
战斗胜利获得奖励
↓
次数每天重置
```

## 七、联盟战斗

### 7.1 集结系统

```
联盟可发起集结
↓
成员响应加入
↓
队伍同时进攻
```

### 7.2 集结流程

```
盟主/副盟主发起集结
↓
选择目标(关卡/城池)
↓
成员响应(5分钟内)
↓
时间到后同时进攻
↓
根据伤害排名奖励
```

### 7.3 快速集结

```
占领城池后
↓
可在城池快速集结
↓
进攻附近关卡/城池
```

## 八、数据结构

### 8.1 地图数据

```java
class MapData {
    Integer regionId;     // 区域ID
    String name;         // 区域名称
    Integer level;       // 区域等级
    List<Waypoint> waypoints; // 坐标点
    List<City> cities;   // 城池
    List<Checkpoint> checkpoints; // 关卡
    List<Ruins> ruins;   // 遗迹
}
```

### 8.2 队伍数据

```java
class Army {
    Long id;
    Long playerId;
    
    List<Long> partnerIds;  // 上阵武将
    
    Integer state;         // 0=待命, 1=行军, 2=战斗, 3=挂机
    
    Integer targetX;      // 目标X坐标
    Integer targetY;      // 目标Y坐标
    
    Integer speed;        // 行军速度
    
    LocalDateTime arriveTime; // 到达时间
    
    // 挂机
    Boolean autoPatrol;  // 是否挂机
    Integer patrolRange; // 巡逻范围
}
```

### 8.3 遗迹记录

```java
class RuinsRecord {
    Long playerId;
    String ruinsType;    // 山洞/古墓/仙府
    
    Integer todayCount;  // 今日次数
    LocalDateTime lastReset; // 重置时间
}
```

## 十、总结

### 核心功能

| 功能 | 说明 |
|------|------|
| 行军 | 点击移动/30%遇怪 |
