## 五、抽卡系统 (5+1)

### 5.1 5+1模式
```
玩家选择5个武将 + 系统随机1个武将 = 6人阵容
```

### 5.2 保底机制
```
┌─────────────────────────────────────────┐
│  20抽必出橙将                                  │
│  累计6个橙将 → 第6次必定出现红色武将           │
└─────────────────────────────────────────┘
```

### 5.3 基础概率
| 品质 | 概率 |
|------|------|
| 红 | 1% |
| 橙 | 5% |
| 紫 | 24% |
| 蓝 | 70% |

### 5.4 保底计算示例
```
第1-19抽: 随机抽取
第20抽: 必出橙将 (保底计数+1)
第21-39抽: 随机抽取
第40抽: 必出橙将 (保底计数+2)
...
第100抽: 6个橙将保底 → 必出红将!
```

### 5.5 名将加成
```
名将加成 = 抽卡时额外获得优质武将的概率

示例:
- 有神将(+5%): 红将概率 = 1% + 5% = 6%
- 有名将(+3%): 红将概率 = 1% + 3% = 4%
- 有良将(+1%): 红将概率 = 1% + 1% = 2%

注: 名将加成只在抽卡时触发，不影响内政系统
```

## 八、战力公式

```
战力 = (武力+智力+统率)×2 + 速度 + 兵量/100 + 技能加成
技能加成 = 技能数×50 + 技能效果值
```

## 九、总结

### 系统特点
1. **统一100级** - 所有武将都可升到100级
2. **名将系统** - 抽卡概率加成，激励收集
3. **5+1抽卡** - 6人阵容，6抽保底出红
4. **差距适中** - 3倍左右，可培养追赶

- 收集名将提升抽卡概率
- 6星武将获得双倍属性
