# 封神榜 v3.1 武将系统重构规划

> 参考《三国志战略版》深度设计

## 一、武将属性系统

### 1.1 六维基础属性
| 属性 | 说明 | 战斗影响 |
|------|------|----------|
| 武力 | 物理攻击 | 物理伤害输出 |
| 智力 | 法术攻击 | 法术伤害+法防 |
| 速度 | 行动顺序 | 先手/后手 |
| 统率 | 队伍指挥 | 物理防御+全队加成 |
| 魅力 | 社交影响 | 抽卡概率 |
| 政令 | 内政建设 | 资源产出 |

### 1.2 战斗伤害公式
```
物理伤害 = 攻击方武力 - 防御方统率 (最低1)
法术伤害 = 攻击方智力 - 防御方智力 (最低1)
物理防御 = 统率值
法术防御 = 智力值
```

### 1.3 兵量系统（替代血量）
> 兵量由玩家等级决定，与武将无关

| 概念 | 说明 |
|------|------|
| 兵量 | 玩家等级决定，初始10000 |
| 等级公式 | 兵量 = 10000 + 玩家等级 × 1000 |
| 死兵 | 战损不可恢复 |
| 伤兵 | 战斗后可恢复 |

### 1.4 成长系统
```
升级属性 = 基础属性 + (等级-1) * 成长值
```

### 1.5 兵种系统
| 兵种 | 特点 | 克制 |
|------|------|------|
| 步兵 | 高防御 | 被骑兵克制 |
| 骑兵 | 高机动 | 克制弓兵 |
| 弓兵 | 远程输出 | 克制步兵 |
| 策士 | 法术伤害 | 需保护 |

## 二、战法系统设计（参考三国志战略版）

### 2.1 战法类型

| 类型 | 触发方式 | 例子 |
|------|----------|------|
| 主动战法 | 战斗中手动释放 | 火尖枪、风雷棍 |
| 被动战法 | 常驻生效 | 护体神光、天赋 |
| 指挥战法 | 战斗开始时 | 鼓舞全军 |
| 联动战法 | 与特定武将组合 | 父子同心 |

### 2.2 战法效果

| 效果 | 说明 |
|------|------|
| 伤害 | 单体/群体攻击(兵量) |
| 恢复 | 恢复兵量 |
| 增益 | 提升属性/增伤 |
| 减益 | 降低属性/易伤 |
| 控制 | 眩晕/混乱/沉默 |
| 护盾 | 吸收伤害 |
| 规避 | 一定概率闪避 |

### 2.3 战法数据结构
```javascript
skill: {
  id: 1,
  name: "火尖枪",
  type: "主动", // 主动/被动/指挥/联动
  target: "单体", // 单体/群体/自身/友军
  cost: { mp: 30 }, // 消耗
  cd: 2, // 冷却2回合
  effect: {
    type: "伤害",
    value: 150 // 150%武力
  }
}
```

### 2.4 战法学习
- 每个武将可携带1-2个战法
- 战法可升级提升效果
- 相同战法不可重复携带

## 三、抽卡系统设计（5+1模式）

### 3.1 5+1模式说明
- **5**: 玩家从卡池选择5个武将
- **1**: 系统随机1个武将
- **6个橙色普通武将必出1个红色武将**

### 3.2 抽卡机制
```
累计抽取6个橙色武将 ↓
第6次必定出现红色武将（隐藏于橙色中）
```

### 3.3 卡池类型

| 卡池 | 概率 | 说明 |
|------|------|------|
| 限时UP | 橙5%/红2% | 活动期间 |
| 常驻池 | 橙3%/红1% | 永久 |
| 友情池 | 紫10%/蓝20% | 免费 |

### 3.4 保底机制
- 10连抽必出橙将及以上
- 累计6抽必出红将（核心保底）

### 3.5 抽卡资源
| 道具 | 获取 | 用途 |
|------|------|------|
| 灵气 | 签到/任务 | 抽卡 |
| 金币 | 战斗/任务 | 升级 |
| 玉符 | 付费 | 限时池 |

## 四、武将体系（100+武将）

### 4.1 等级显示规则
- **显示最高**: 橙色（SR）
- **隐藏红将**: 红色武将显示为橙色，但实际属性为红将
- **保底机制**: 6抽橙将必出红将（隐藏）

### 4.2 武将品质
| 显示品质 | 实际品质 | 说明 |
|-----------|----------|------|
| 橙色 | 橙色(SR) | 常规橙将 |
| 橙色 | 红色(SSR) | 隐藏红将，保底产出 |
| 紫色 | 紫色(R) | 常规紫将 |
| 蓝色 | 蓝色(N) | 基础将 |

### 4.3 武将数量规划（100+）
