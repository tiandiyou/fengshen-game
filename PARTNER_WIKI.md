## 一、平衡性算法

### 1.1 基础属性公式

```
武力 = 基础武力 × (1 + 品质系数) + 随机波动(-5~+5)
智力 = 基础智力 × (1 + 品质系数) + 随机波动(-5~+5)
统率 = 基础统率 × (1 + 品质系数) + 随机波动(-5~+5)
速度 = 基础速度 × (1 + 品质系数) + 随机波动(-3~+3)
```

### 1.2 品质系数

| 品质 | 系数 | 属性加成 |
|------|------|----------|
| 红(SSR) | 0.5 | +50% |
| 橙(SR) | 0.3 | +30% |
| 紫(R) | 0.15 | +15% |
| 蓝(N) | 0 | 基准值 |

### 1.3 成长值公式

```
成长值 = 基础成长 × 品质系数
```

| 品质 | 武力成长 | 智力成长 | 统率成长 |
|------|----------|----------|----------|
| 红(SSR) | 6-10 | 4-8 | 4-7 |
| 橙(SR) | 4-7 | 2-5 | 2-5 |
| 紫(R) | 2-4 | 1-3 | 1-3 |
| 蓝(N) | 1-2 | 0-1 | 0-1 |

### 1.4 技能点公式

| 品质 | 技能点数 | 技能效果加成 |
|------|----------|--------------|
| 红(SSR) | 2 | +50% |
| 橙(SR) | 1-2 | +30% |
| 紫(R) | 1 | +15% |
| 蓝(N) | 0 | 0 |

### 1.5 战力计算公式

```
战力 = (武力 + 智力 + 统率) × 2 + 速度 + 兵量/100 + 技能加成
技能加成 = 技能数 × 50 + 技能效果值
```
