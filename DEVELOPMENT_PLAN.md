## 📊 技术实现

### 后端API

```
地图系统:
- GET    /api/map/zones           获取区域列表
- GET    /api/map/my              我的位置
- POST   /api/map/move            移动到区域
- POST   /api/map/relic           探索遗迹
- POST   /api/map/transfer         传送

城战:
- POST   /api/war/siege           攻城
- POST   /api/war/defend          防守
- GET    /api/war/cities          城池列表
- GET    /api/war/report/{id}     战报

同盟:
- POST   /api/alliance/create    创建
- POST   /api/alliance/join       加入
- POST   /api/alliance/leave      退出
- GET    /api/alliance/list       列表
- POST   /api/alliance/war       宣战
- POST   /api/alliance/diplomacy 外交
```
