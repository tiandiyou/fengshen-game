
## 三、综合战力计算

### 3.1 完整战力公式

```
战力 = 基础战力 + 等级成长 + 星级加成 + 技能加成

基础战力 = (武力 + 智力 + 统率) × 2 + 速度
等级成长 = (等级 - 1) × (成长武力 + 成长智力 + 成长统率) × 2
星级加成 = 基础战力 × 星级百分比
技能加成 = 技能数 × 50 + 技能效果值
```

### 3.2 战力计算示例

**哪吒 80级 6星**:
```
基础战力 = (160+75+80)×2 + 90 = 630
等级成长 = 79 × (3+2.5+2)×2 = 79 × 15 = 1185
星级加成 = 630 × 100% = 630
技能加成 = 2 × 50 + 80 = 180

总战力 = 630 + 1185 + 630 + 180 = 2625
```

**金吒 60级 6星**:
```
基础战力 = (130+45+68)×2 + 78 = 520
等级成长 = 59 × (2+1.5+1.2)×2 = 59 × 9.2 = 542.8
星级加成 = 520 × 100% = 520
技能加成 = 1 × 50 + 35 = 85

总战力 = 520 + 543 + 520 + 85 = 1668
```
