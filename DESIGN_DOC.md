## 一、伤害系统

### 1.1 物理伤害公式 (方案B)

```
物理伤害 = 武力 × (1 - 统率 / (统率 + 200))
```

**示例**:
| 武力 | 统率 | 伤害 | 减伤率 |
|------|------|------|----------|
| 180 | 0 | 180 | 0% |
| 180 | 50 | 144 | 20% |
| 180 | 180 | 95 | 47% |
| 180 | 500 | 52 | 71% |

### 1.2 法术伤害公式

```
法术伤害 = 智力 × (1 - 敌方智力 / (敌方智力 + 200))
```

### 1.3 最低伤害

```
最低伤害 = 1 (即使防御极高也有输出)
```

## 四、抽卡系统

### 4.1 5+1模式

```
玩家选择5个武将 + 系统随机1个武将 = 6人阵容
```

### 4.2 保底机制

```
┌─────────────────────────────────────────┐
│  20抽 → 必出橙将                                  │
│  累计6个橙将 → 必出红色武将                       │
└─────────────────────────────────────────┘
```

### 4.3 基础概率

| 品质 | 概率 |
|------|------|
| 红 | 1% |
| 橙 | 5% |
| 紫 | 24% |
| 蓝 | 70% |

### 4.4 名将加成

| 名将 | 红将概率 |
|------|-----------|
| 无 | 1% |
| 良将(+1%) | 2% |
| 名将(+3%) | 4% |
| 神将(+5%) | 6% |

### 4.5 抽卡流程

```
1. 玩家选择5个想要的武将
2. 系统随机生成1个武将
3. 20抽内必出1个橙将
4. 累计6个橙将后，下一个橙将必定是红将
```

## 五、等级与成长

### 5.1 等级上限

```
所有武将统一最高100级
```

### 5.2 升级经验

```
升级经验 = 等级 × 150
```

### 5.3 成长值

| 品质 | 武力/级 | 智力/级 | 统率/级 | 速度/级 |
|------|---------|---------|---------|---------|
| 红 | 2.0 | 2.0 | 1.5 | 1.0 |
| 橙 | 1.6 | 1.6 | 1.2 | 0.8 |
| 紫 | 1.2 | 1.2 | 0.9 | 0.6 |
| 蓝 | 0.8 | 0.8 | 0.6 | 0.4 |

### 5.4 属性公式

```
当前属性 = 基础属性 + (等级 - 1) × 成长值
```

### 5.5 100级属性示例

```
红将哪吒 (100级):
- 武力 = 160 + 99×2.0 = 358
- 智力 = 75 + 99×1.5 = 223
- 统率 = 80 + 99×1.0 = 179
- 速度 = 90 + 99×0.8 = 169

橙将金吒 (100级):
- 武力 = 130 + 99×1.6 = 288
- 智力 = 45 + 99×1.2 = 164
- 统率 = 68 + 99×0.9 = 157
- 速度 = 78 + 99×0.6 = 137
```

## 六、星级系统

### 6.1 星级加成

| 星级 | 属性加成 | 升级消耗 |
|------|----------|----------|
| 1星 | +0% | - |
| 2星 | +15% | 2个相同 |
| 3星 | +30% | 2个相同 |
| 4星 | +50% | 2个相同 |
| 5星 | +75% | 2个相同 |
| 6星 | +100% | 2个相同 |

### 6.2 满星消耗

```
6星总共需要: 2+2+2+2+2 = 10个相同武将
```

### 6.3 星级属性公式

```
实际属性 = 基础属性 × (1 + 星级加成)
```

## 八、战力计算

### 8.1 战力公式

```
战力 = (武力 + 智力 + 统率) × 2 + 速度 + 兵量/100 + 技能加成

技能加成 = 技能数 × 50 + 技能效果值
```

### 8.2 差距控制

| 对比 | 1级差距 | 100级差距 |
|------|----------|------------|
| 红 vs 橙 | ~1.5倍 | ~2倍 |
| 红 vs 蓝 | ~2倍 | ~3倍 |

## 九、总结

### 核心系统

1. **伤害系统**: 物理/法术公式B，百分比穿透
2. **技能系统**: 5大类，100+技能
3. **武将系统**: 100武将，品质/名将/星级
4. **抽卡系统**: 5+1模式，20抽保底，6橙保底
5. **等级成长**: 统一100级，星级加成

### 玩家目标

- 收集名将提升抽卡概率
- 培养武将到100级
- 6星武将获得双倍属性
- 挑战更高章节
